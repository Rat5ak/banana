## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open your browser to `http://localhost:3000` and start collecting bananas!

## Leaderboard

Enter a name in the field on the page and click **Submit Score** to record
your current number of collected bananas. The top scores are displayed in the
leaderboard list.
